Radon UI

一个用于企业级后台产品开发的简洁易用的Vue组件库

特性

  • 基于 Vue 开发的高质量UI组件
  • 基于 npm + webpack + ES6 + postcss开发
  • 数据驱动，简单易使用

示例

<template>
    <div class="customer">
        <rd-date-picker :date="startDate"></rd-date-picker>
    </div>
</template>

<script>
    import 'radon-ui/dist/index.css'
    import { rdDatePicker } from 'radon-ui'
    export default {
        data () {
            return {
                startDate: {
                    value: ''
                }
            }
        },
        components: {
            rdDatePicker
        }
    }
</script>
